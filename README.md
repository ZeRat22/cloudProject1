# CloudNexus | Enterprise Cloud Solutions

CloudNexus is a modern, responsive landing page and comparison platform for enterprise cloud solutions. It provides a comprehensive overview of major cloud providers (AWS, Azure, and GCP) while showcasing CloudNexus's own high-performance infrastructure services.

## 🚀 Features

- **Comparison Engine:** Detailed breakdown of AWS, Azure, and GCP market share, strengths, and use cases.
- **Modern UI/UX:** Clean, professional design with high-quality visuals and typography.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Environment Detection:** Real-time detection and display of the user's browser and operating system.
- **Interactive Elements:** Smooth navigation, mobile menu toggle, and icon-rich interface using Lucide Icons.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for optimal SEO and accessibility.
- **CSS3:** Custom styling with CSS variables, Flexbox, and Grid.
- **JavaScript:** Client-side environment detection and mobile interactivity.
- **Lucide Icons:** Modern, consistent iconography.
- **Google Fonts:** Inter font family for high legibility.
